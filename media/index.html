CLI profesional para bases de datos

Gestiona, migra y valida bases de datos desde una sola herramienta.

DBAdmin reúne administración relacional, soporte NoSQL, migración ETL y validación de backups en Docker. Esta landing sirve como presentación, documentación y punto de descarga del ejecutable.

Documentación

Resumen práctico de los comandos principales para demo y presentación.

Modo relacional
Conectar a MySQL dbcli-rel > connect mysql olimpiadas_upt root 75236233
Consultar tablas dbcli-rel > show tables
CRUD rápido dbcli-rel > select * from usuarios | insert into usuarios (...) values (...)
Importar un backup dbcli-rel > import_db C:/aaa/olimpiadas_upt.sql
Migrar a SQLite o PostgreSQL dbcli-rel > migrate C:/aaa/olimpiadas_upt.sql sqlite C:/aaa/olimpiadas_upt.db
Validar backup dbcli-rel > validate backup C:/aaa/olimpiadas_upt.sql mysql olimpiadas_upt
Más funcionalidades
# Ejecutar el proyecto
cd src
python main.py

# Exportar la base completa
